# AI-COVER Using RVC V2

- Link: https://github.com/SociallyIneptWeeb/AICoverGen

1. Clone & download model
```# command
git clone https://github.com/liemthanh-playgroundvina/ai-cover
cd ai-cover
make download_model
```

2. Build Image
```# command
make build
```
3. Start
```# command
make start
```
